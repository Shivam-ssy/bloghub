# BlogHub

BlogHub is a simple blogging platform built with **Django** where users can read blogs and authenticated users can create, edit, and delete their own posts.

This was my **first Django project**, created to learn Django fundamentals like project structure, authentication, CRUD operations, templates, and deployment.

---

## 🚀 Live Demo

Live Project: [click here ](https://bloghub-uhf5.onrender.com/)
GitHub Repository: [click here](https://github.com/Shivam-ssy/bloghub)

---

## ✨ Features

- 📖 Anyone can read blog posts
- ✍️ Authenticated users can create posts
- 📝 Edit and delete blog posts
- 🏷 Posts include title, description, tags, and image
- 🖼 Image upload support
- 🔐 User authentication (login/signup)
- 📂 Organized templates and static files
- ☁️ Media storage support

---

## 🛠 Tech Stack

- **Backend:** Django
- **Database:** SQLite (development) / PostgreSQL (production)
- **Frontend:** HTML, CSS, Django Templates
- **Media Storage:** Cloudinary
- **Deployment:** Render



## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Shivam-ssy/bloghub.git
cd bloghub
Create virtual environment
python -m venv venv
```
Activate it:

Linux / Mac
```
source venv/bin/activate
```
Windows
```
venv\Scripts\activate
```
3️⃣ Install dependencies
```
pip install -r requirements.txt
```

4️⃣ Run migrations
```
python manage.py migrate
```
5️⃣ Run development server
```
python manage.py runserver

```
Open in browser:
```
http://127.0.0.1:8000
```
